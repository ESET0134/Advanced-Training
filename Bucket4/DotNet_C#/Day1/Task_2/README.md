### Simple Salary Computation

Description:

    Create a program to calculate the net salary of an employee using variables.

Steps:
1. Declare variables for:
    1. Basic salary
    2. HRA (20% of basic)
    3. DA (10% of basic)
    4. Tax (8% of gross)
2. Use formulas: gross=basic +HRA+DA
netSalary=gross-tax
3. Display all values with proper labels.
